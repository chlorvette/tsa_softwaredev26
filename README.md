This is an online course platform designed with built-in accessibility features to minimize barriers for those with visual disabilities. Users can adjust their preferences to what works best for them, which are saved to their account. They can change and delete their account at any time, and important visual markers on the website are clearly marked and highlighted for accessibility. 

## Running the application:
```
flask --debug run
```

## Running the tests (checks that each route returns 200):
```
pytest
```
